# Time Series Analysis and Its Applications

Welcome to this interactive textbook on time series analysis. This book is based on lecture materials from a course at the University of Maryland, College Park given in the fall semester of 2025. 

## About This Book

This textbook covers:
- Fundamental concepts in time series analysis
- Mathematical foundations (complex numbers, geometric series)
- ARMA and SARMA models
- Forecasting techniques
- Signal processing and filtering
- Practical implementation in Python

## Navigation

Use the sidebar to navigate through chapters. Each chapter includes:
- Theory and mathematical foundations
- Interactive Python examples
- Real-world applications
- Exercises and problems

## Prerequisites

This course assumes familiarity with:
- Probability and statistics
- Linear algebra
- Basic calculus
- Python programming
